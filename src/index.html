¿Que es Octave?

GNU Octave es un lenguaje de alto nivel, destinado principalmente a cálculos numéricos. Proporciona una interfaz de línea de comandos conveniente para resolver problemas lineales y no lineales numéricamente, y para realizar otros experimentos numéricos utilizando un lenguaje que es principalmente compatible con Matlab. También se puede usar como un lenguaje orientado a lotes.

Octave tiene amplias herramientas para resolver problemas comunes de álgebra lineal numérica, encontrar las raíces de ecuaciones no lineales, integrar funciones ordinarias, manipular polinomios e integrar ecuaciones diferenciales y algebraicas diferenciales ordinarias. Es fácilmente extensible y personalizable a través de funciones definidas por el usuario escritas en el propio lenguaje de Octave, o utilizando módulos cargados dinámicamente escritos en C ++, C, Fortran u otros idiomas.

¿Donde conseguirlo?

Octave es software libre. Esto significa que todos son libres de usarlo y redistribuirlo bajo ciertas condiciones. Sin embargo, Octave no es de dominio público. Tiene derechos de autor y existen restricciones en su distribución, pero las restricciones están diseñadas para garantizar que otros tengan la misma libertad para usar y redistribuir Octave que usted tiene. Las condiciones precisas se pueden encontrar en la Licencia Pública General de GNU que viene con Octave y que también aparece en copia.

Las distribuciones individuales proporcionan versiones ejecutables de GNU Octave para sistemas GNU/Linux. Las distribuciones conocidas para empaquetar Octave incluyen Debian, Ubuntu, Fedora, Gentoo y openSUSE. Estos paquetes son creados por voluntarios. El retraso entre un lanzamiento de origen de Octave y la disponibilidad de un paquete para una distribución particular de GNU/Linux varía.

Crear una matriz

Los vectores y las matrices son los bloques de construcción básicos para el análisis numérico. Para crear una nueva matriz y almacenarla en una variable para que pueda consultarla más tarde.

  1. A = [1,2,3,4; 5,6,7,8]: Matriz 2x4
  2. zeros(n): Matriz nxn de zeros
  3. eye(n): Matriz idententidad nxn
  4. inv(A): Inversa Matriz A
  5. det(A): Determinante Matriz A

Octave responderá imprimiendo la matriz en columnas perfectamente alineadas. Octave usa una coma o un espacio para separar las entradas en una fila, y un punto y coma o un retorno de carro para separar una fila de la siguiente. Finalizar un comando con un punto y coma le dice a Octave que no imprima el resultado del comando.

Operaciones con matrices

Octave tiene una notación de operador conveniente para realizar aritmética de matrices. Por ejemplo, para multiplicar la matriz A por un valor escalar.

2 * A

Para multiplicar las dos matrices A y B.

A * B

Para formar la matriz de transposición del producto (A) * A.

A' * A

Crear un gráfico

Para mostrar gráficamente la solución del ejemplo anterior, use el comando:

plot (t, x)

Si está utilizando una interfaz gráfica de usuario, Octave creará automáticamente una ventana separada para mostrar el gráfico.
Para guardar un diagrama una vez que se ha mostrado en la pantalla, use el comando de impresión. Por ejemplo:

print -dpdf foo.pdf

Creará un archivo llamado foo.pdf que contiene una representación de la trama actual en formato de documento portátil. El comando

help print

Explica más opciones para el comando de impresión y proporciona una lista de formatos de archivo de salida adicionales.
